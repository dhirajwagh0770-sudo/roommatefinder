Project Title : " Roommate Finder "

Project Description :
The Roommate finder Website is a web-based platform developed to help users 
find suitable roommates and available rooms easily and quickly. In many cities, 
students and working professionals face difficulties in finding shared 
accommodation, trusted roommates, and verified room details. Traditional 
methods such as brokers, social media posts, and manual searching are time
consuming and often unreliable.
